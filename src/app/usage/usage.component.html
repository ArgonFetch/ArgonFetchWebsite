How to Use ArgonFetch

Once you have ArgonFetch installed and running (usually at http://localhost:8080), using it is straightforward:

1. Paste the Media URL

Navigate to ArgonFetch in your web browser. You'll see an input field. Simply paste the URL of the video, song, or media you wish to download.

2. Choose Options (If Available)

Depending on the source and media type, you might be presented with options to select quality, format, or other preferences.

3. Start Downloading

Once you've set your preferences, click the download button. ArgonFetch will process your request and begin downloading the media.

Tip: Keep an eye on the Roadmap section for upcoming features and supported platforms!
