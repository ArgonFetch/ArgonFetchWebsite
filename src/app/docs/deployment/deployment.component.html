Deployment

Learn how to deploy ArgonFetch to production environments.

Docker Deployment

The easiest way to deploy ArgonFetch is using Docker:

1. Build Docker Image

docker build -t argonfetch .

2. Run Container

docker run -p 3000:3000 -v ./downloads:/app/downloads argonfetch

Manual Deployment

For manual deployment on a server:

1. Build for Production

npm run build

2. Start Production Server

NODE_ENV=production npm start

Environment Configuration

Make sure to set these environment variables in production:

NODE_ENV=production
PORT=3000
HOST=0.0.0.0

Reverse Proxy Setup

Example Nginx configuration:

server {{ '{' }}
    listen 80;
    server_name your-domain.com;

    location / {{ '{' }}
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    {{ '}' }}
{{ '}' }}
