Usage Guide

Learn how to use ArgonFetch to download media from Spotify, YouTube, and other platforms.

Web Interface

The simplest way to use ArgonFetch is through the web interface:

1. Access the Web Interface

Open your browser and navigate to your ArgonFetch installation:

http://localhost:8080

Or use the cloud version at app.argonfetch.dev

2. Enter Media URL

Paste the URL of the media you want to download:

  • Spotify song URL (e.g., https://open.spotify.com/track/...) - Searches and downloads from YouTube Music
  • YouTube video URL (e.g., https://youtube.com/watch?v=...)
  • SoundCloud track URL (e.g., https://soundcloud.com/artist/track)
  • TikTok video URL (e.g., https://www.tiktok.com/@user/video/...)
  • Other supported platform URLs

3. Select Format

Choose your preferred output format:

  • MP3 - Audio only, widely compatible
  • MP4 - Video with audio
  • WebM - Open format, good compression

4. Download

Click the "Download" button and wait for the process to complete. Your file will be automatically downloaded to your browser's download folder.

API Usage

ArgonFetch provides a REST API for programmatic access. API documentation is available at:

http://localhost:8080/swagger

Example: Download a Spotify Song

curl -X POST http://localhost:8080/api/download \
  -H "Content-Type: application/json" \
  -d '{{ '{' }}
    "url": "https://open.spotify.com/track/4iV5W9uYEdYUVa79Axb7Rh",
    "format": "mp3"
  {{ '}' }}'

Example: Download a YouTube Video

curl -X POST http://localhost:8080/api/download \
  -H "Content-Type: application/json" \
  -d '{{ '{' }}
    "url": "https://youtube.com/watch?v=dQw4w9WgXcQ",
    "format": "mp4"
  {{ '}' }}'

Tips & Best Practices

  • Always ensure you have the right to download content
  • For Spotify downloads, make sure your API credentials are properly configured
  • Use MP3 format for audio-only content to save bandwidth and storage
  • Check the API documentation at /swagger for advanced features
