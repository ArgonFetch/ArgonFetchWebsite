Basic Usage

Learn how to use ArgonFetch to download media from various platforms quickly and efficiently.

Web Interface

The simplest way to use ArgonFetch is through the web interface:

1. Access the Web Interface

Open your browser and navigate to your ArgonFetch installation:

http://localhost:3000

2. Enter Media URL

Paste the URL of the video or media you want to download in the input field.

3. Select Quality (Optional)

Choose your preferred quality settings from the dropdown menu.

4. Download

Click the "Download" button and wait for the process to complete.

API Usage

For programmatic access, you can use the REST API:

Basic Download Request

curl -X POST http://localhost:3000/api/download \
  -H "Content-Type: application/json" \
  -d '{{ '{' }}
    "url": "https://example.com/video",
    "format": "mp4",
    "quality": "720p"
  {{ '}' }}'

Check Download Status

curl http://localhost:3000/api/status/abc123

Quality Options

Available quality settings depend on the source platform:

Quality Resolution Use Case
best Highest available Best quality download
1080p 1920x1080 Full HD
720p 1280x720 HD (recommended)
480p 854x480 Standard quality
worst Lowest available Fastest download

Tips and Best Practices

  • Check platform policies: Always respect the terms of service of the platforms you're downloading from
  • Storage space: Higher quality videos require more storage space
  • Network bandwidth: Consider your internet connection when selecting quality
  • Batch downloads: Use the API for downloading multiple files

Troubleshooting

Download fails immediately

Make sure the URL is valid and the media is publicly accessible.

Slow download speeds

Try selecting a lower quality or check your network connection.

Unsupported platform

Check our API documentation for the list of supported platforms.
