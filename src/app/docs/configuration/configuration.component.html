Configuration

Learn how to configure ArgonFetch to suit your needs and preferences.

Environment Variables

ArgonFetch can be configured using the following environment variables:

PORT

The port on which the server will run.

PORT=3000

NODE_ENV

The environment mode (development, production).

NODE_ENV=production

Configuration File

You can also use a configuration file to customize ArgonFetch behavior:

{{ '{' }}
  "server": {{ '{' }}
    "port": 3000,
    "host": "localhost"
  {{ '}' }},
  "downloads": {{ '{' }}
    "outputPath": "./downloads",
    "maxConcurrent": 3
  {{ '}' }}
{{ '}' }}
