API Reference

Complete reference for the ArgonFetch API endpoints and methods.

Download Endpoint

POST /api/download

Download media from a supported platform.

Request Body

{{ '{' }}
  "url": "https://example.com/video",
  "format": "mp4",
  "quality": "720p"
{{ '}' }}

Response

{{ '{' }}
  "success": true,
  "downloadId": "abc123",
  "filename": "video.mp4",
  "size": 15728640
{{ '}' }}

Status Endpoint

GET /api/status/:id

Check the status of a download.

Response

{{ '{' }}
  "id": "abc123",
  "status": "completed",
  "progress": 100,
  "downloadUrl": "/downloads/video.mp4"
{{ '}' }}
