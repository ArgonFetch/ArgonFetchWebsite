Installation Guide

ArgonFetch can be easily installed and set up on your system. Follow the steps below to get started with self-hosting ArgonFetch.

1 Prerequisites

Before installing ArgonFetch, make sure you have the following prerequisites installed on your system:

  • Node.js (version 18 or higher)
  • npm or yarn package manager
  • Git for cloning the repository

2 Clone the Repository

Start by cloning the ArgonFetch repository from GitHub:

git clone https://github.com/ArgonFetch/ArgonFetch.git
cd ArgonFetch

3 Install Dependencies

Install the required Node.js dependencies:

npm install

Or if you prefer using Yarn:

yarn install

4 Configuration

Copy the example environment file and configure your settings:

cp .env.example .env

Edit the .env file with your preferred settings:

Note: Make sure to configure the PORT and other environment variables according to your needs.

5 Start the Application

Start ArgonFetch in development mode:

npm run dev

For production deployment:

npm run build
npm start

ArgonFetch will be available at http://localhost:3000 (or your configured port).

6 Docker Installation (Alternative)

You can also run ArgonFetch using Docker:

docker build -t argonfetch .
docker run -p 3000:3000 argonfetch

Or use Docker Compose:

docker-compose up -d
