Self-Host ArgonFetch

Follow these steps to get your own instance of ArgonFetch running locally.

1 Prerequisites

For Spotify support, you'll need to create an App using Spotify for Developers.

From there, obtain a Client ID and a Client Secret. You will be prompted for these during the setup process.
