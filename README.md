## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone this repository
   ```sh
   git clone https://github.com/ArgonFetch/ArgonFetchWebsite.git
   cd ArgonFetchWebsite
   ```

2. Install dependencies
   ```sh
   npm install
   ```

3. Start the development server
   ```sh
   npm start
   ```

4. Open your browser at `http://localhost:4200`
